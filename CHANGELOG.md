# Changelog

## Unreleased

- 요소 이동·프레임 크기·레이아웃 수치·프리셋·초기화 직전 상태를 프레임 스타일별 최대 50단계 복원하는 `돌아가기` 버튼 추가, 인스탁스 우상단 그립은 왼쪽·아래로 끌면 여백이 커지도록 방향 반전
- 기본 시스템 글꼴과 분리된 PC 설치 글꼴 선택(데스크톱 Chromium) 및 TTF·OTF·WOFF·WOFF2 파일 글꼴 추가 — 영문·한글을 문자권별로 렌더하고, 사용 중인 파일 글꼴은 용량 검사 후 프로젝트에 포함해 복원
- 프레임 스타일마다 세로·가로 레이아웃을 독립 저장해 스타일 전환 시 여백·요소 위치가 섞이지 않도록 수정하고, 가로 인스탁스 회전 글씨의 개별 이동 지원
- 하단 사진 스트립의 가로·세로 썸네일을 원본 비율에 맞춰 사진 전체가 보이도록 개선
- 첫 화면 안내문의 폭과 줄바꿈 균형을 조정해 짧은 어절이 홀로 다음 줄에 남지 않게 개선
- 사진 순서·설정·장별 EXIF 수정값을 사진 원본 없이 `.photoframe.json`으로 저장하고, 원본 재선택으로 복원하는 프로젝트 파일 추가
- 테두리 여백(인스탁스) 조정을 사진 우상단 모서리 그립으로 변경
- 조정 부위 호버 강조 + 방향 커서 (박스 이동/모서리/선)
- 경계 표시 기본 켜짐 — 줌바 '경계'로 끄면 선택이 기억된다

## v1.0.0 — 2026-07-14

첫 공개 릴리즈. 프로토타입(v3 단일 HTML)에서 출발해 실사용 수준으로 확장.

### 프레임
- 필름 스트립 / 인스탁스 / 하단·상단 여백 4종
- 사진 방향 자동 대응 — 세로 사진은 필름이 세로로(퍼포레이션 좌우), 가로 사진 인스탁스는 우측 여백 카드(글씨 가로 스택/세로 회전 선택)
- 테두리 여백·하단 공간·글자 크기 배율, 글꼴 4종(시스템 폰트)

### 촬영 정보
- 카메라/렌즈/필름·기타/날짜/설정값(F·셔터·ISO·초점거리)/여러 줄 메모
- JPEG EXIF 장별 자동 입력 (자체 파서, 스캐너 기기명 가드, EXIF 배지로 원본 복원)
- 항목별 눈 아이콘 표시 토글, 장비 프리셋 팔레트
- 로고·서명 — 이미지 슬롯 + 텍스트 서명(필기체)

### 레이아웃
- 세로/가로 사진 별도 프로파일 (모든 배치 설정)
- 경계 표시 + 드래그 배치 — 수평/수직 스냅 가이드, 더블클릭 초기화

### 내보내기
- 인스타 비율 맞춤 (자동/4:5/1:1/1.91:1/9:16 스토리, 여백 색 선택)
- PNG/JPEG(품질), 긴 변 리사이즈, 폴더 지정 일괄 저장(Chromium)
- 파일 제목 순번 명명 — 제목(1), 제목(2)…

### 안전장치·기타
- EXIF 회전 자동 보정, 브라우저 캔버스 한계 자동 축소
- 모든 설정 localStorage 자동 저장 (사진은 어디에도 저장·전송되지 않음)
- 개인 설정 — 패널 폭·UI 배율
