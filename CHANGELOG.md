# Changelog

## v1.0.0 — 2026-07-14

첫 공개 릴리즈. 프로토타입(v3 단일 HTML)에서 출발해 실사용 수준으로 확장.

### 프레임
- 필름 스트립 / 인스탁스 / 하단·상단 여백 4종
- 사진 방향 자동 대응 — 세로 사진은 필름이 세로로(퍼포레이션 좌우), 가로 사진 인스탁스는 우측 여백 카드(글씨 가로 스택/세로 회전 선택)
- 테두리 여백·하단 공간·글자 크기 배율, 글꼴 4종(시스템 폰트)

### 촬영 정보
- 카메라/렌즈/필름·기타/날짜/설정값(F·셔터·ISO·초점거리)/여러 줄 메모
- JPEG EXIF 장별 자동 입력 (자체 파서, 스캐너 기기명 가드, EXIF 배지로 원본 복원)
- 항목별 눈 아이콘 표시 토글, 장비 프리셋 팔레트
- 로고·서명 — 이미지 슬롯 + 텍스트 서명(필기체)

### 레이아웃
- 세로/가로 사진 별도 프로파일 (모든 배치 설정)
- 경계 표시 + 드래그 배치 — 수평/수직 스냅 가이드, 더블클릭 초기화

### 내보내기
- 인스타 비율 맞춤 (자동/4:5/1:1/1.91:1/9:16 스토리, 여백 색 선택)
- PNG/JPEG(품질), 긴 변 리사이즈, 폴더 지정 일괄 저장(Chromium)
- 파일 제목 순번 명명 — 제목(1), 제목(2)…

### 안전장치·기타
- EXIF 회전 자동 보정, 브라우저 캔버스 한계 자동 축소
- 모든 설정 localStorage 자동 저장 (사진은 어디에도 저장·전송되지 않음)
- 개인 설정 — 패널 폭·UI 배율
