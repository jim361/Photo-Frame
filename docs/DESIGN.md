# FilmFrame — 디자인 시스템

## 컨셉: 암실 라이트테이블

UI 전체가 "암실에서 라이트테이블 위에 필름을 올려놓고 보는" 경험을 모사한다.

- 왼쪽 넓은 영역이 **라이트테이블**: 어두운 배경(상단에서 떨어지는 은은한 radial-gradient 조명) 위에 합성 결과가 떠 있다.
- 오른쪽 패널이 **리베이트 레일**: 필름 가장자리(rebate)처럼 정보와 컨트롤이 붙는다. 패널 왼쪽 가장자리에 퍼포레이션(스프로킷 홀) 장식이 세로로 반복된다 — 이 프로젝트의 시그니처 요소.
- 하단 썸네일 바가 **필름 스트립**: 롤에서 잘라낸 컷들이 나란히 놓인다.

## 컬러 토큰 (UI, CSS 변수)

| 토큰 | 값 | 용도 |
|---|---|---|
| `--base` | `#14130f` | 페이지 배경 (암실) |
| `--panel` | `#1c1a15` | 우측 패널 배경 |
| `--panel-hi` | `#24211b` | 버튼·세그먼트 등 패널 위 요소 |
| `--rebate` | `#6e6a5f` | 보조 텍스트, 라벨 (필름 리베이트 회색) |
| `--bone` | `#efeae0` | 본문 텍스트 (뼈색 아이보리) |
| `--edge` | `#e8873a` | 강조색 — 필름 엣지 프린트의 주황. 선택 상태, 주 버튼 |
| `--line` | `#322e26` | 구분선, 테두리 |

## 캔버스 렌더링 색상 (JS 상수 `C`)

UI 토큰과 별도로, 합성 결과물에 쓰는 색은 스크립트의 `C` 객체에 있다.

| 키 | 값 | 용도 |
|---|---|---|
| `base` | `#0d0c0a` | 필름 스트립 프레임의 필름 베이스 검정 |
| `edge` | `#e8873a` | 엣지 프린트 주황 (현재 결과물에서는 미사용 — UI 강조색과 쌍) |
| `rebate` | `#8a8378` | 필름 스트립의 장비·필름 텍스트 |
| `paper` | `#f6f3ec` | 인스탁스 카드 아이보리 |
| `ink` | `#2a2723` | 인스탁스 본문 잉크 |
| `bandDark` | `#14130f` | 어두운 여백 띠 |
| `bandDarkText` | `#cfc9bd` | 어두운 띠 위 텍스트 |

퍼포레이션 구멍은 `#050503` (베이스보다 더 깊은 검정 — 빛이 통과하는 구멍).

## 타이포그래피

- **모노스페이스** (`ui-monospace, SF Mono, Menlo, Cascadia Mono, Roboto Mono`): 라벨, 데이터, 버튼, 워드마크 — 필름 엣지 프린트와 장비 각인의 느낌. 캔버스 렌더링의 장비/필름/프레임 번호도 모노스페이스.
- **시스템 산세리프** (`system-ui, Apple SD Gothic Neo, Malgun Gothic`): 본문, 안내 문구. 인스탁스 프레임의 장비 줄만 예외적으로 Helvetica 계열 산세리프(카드에 인쇄된 느낌).
- 라벨은 대문자 + 넓은 letter-spacing(.2em)으로 각인 느낌을 강화.

## 캔버스 렌더링 수치 (모두 사진 폭 W 기준 비율)

### 필름 스트립 (`film`)
- 위 바 높이: `W × 0.088`, 아래 바: 위 바의 **1.75배** (두 줄 레이아웃이면 **2.3배**로 확장)
- 퍼포레이션: 폭 `bar × 0.50`, 높이 `bar × 0.36`, 모서리 반경 `높이 × 0.24`. 아래 줄 구멍은 사진 아래 `bar × 0.70` 고정 위치 — 줄 수와 무관
- 퍼포레이션 개수: `max(6, round(W / (구멍폭 × 2.15)))` — 사진이 넓을수록 늘어나되 최소 6개
- 텍스트: 장비·필름은 오른쪽 정렬·리베이트 회색·`bar × 0.33px`, letter-spacing 적용
  - 한 줄: 기준선 = 아래 바의 82% 지점
  - 두 줄: 장비 줄 기준선 `bar × 1.45`, 필름 줄 `bar × 2.0` (사진 아래 기준)

### 인스탁스 (`instax`)
- 좌우·상단 여백: `W × 0.055`, **하단 여백: `W × 0.27`** — 실제 instax mini 카드(54×86mm, 화면 46×62mm)의 하단 여백 비율에서 유도한 값
- 사진 둘레에 1px 안팎의 옅은 스트로크(`rgba(0,0,0,.13)`) — 카드에 인화된 경계
- 하단 여백 중앙에 장비(산세리프, 잉크색) / 필름명(모노, 회갈색, 대문자) 2단 배치

### 여백 띠 (`bottom` / `top`)
- 띠 높이: 한 줄 `W × 0.078` / 두 줄 `W × 0.112`
- 글자 크기는 줄 수와 무관하게 `W × 0.0234` (한 줄 띠의 30%) — 줄이 늘어도 텍스트가 커지지 않는다
- 밝은 톤 `#f2efe8` / 어두운 톤 `#14130f` 선택
- 장비·필름은 왼쪽 정렬

## 상호작용 원칙

1. **로드는 항상 화면 맞춤.** 사진을 새로 불러오거나 선택하면 전체가 보이는 fit 배율로 시작한다.
2. **타이핑 중에는 줌을 유지한다.** 촬영 정보를 입력하는 동안 사용자가 확대해 둔 배율·위치를 잃지 않는다 (`refresh(keepZoom=true)`).
3. **스타일 변경 시에만 줌을 리셋한다.** 캔버스 비율 자체가 바뀌므로 화면 맞춤으로 돌아가는 것이 자연스럽다.
4. 줌은 커서 고정점 기준(휠), 팬은 드래그. 키보드 `+ − 0 1` 지원.
5. 피드백은 상단이 아닌 **하단 중앙 토스트** (주황 배경) — 라이트테이블 위 시선을 가리지 않는다.
